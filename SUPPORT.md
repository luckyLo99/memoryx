## Reporting Issues

Before reporting an issue:

1. Check the [documentation](docs/) for existing answers
2. Search [existing issues](https://github.com/luckyl214/memoryx/issues)
3. Run `memoryx doctor` and include the output
4. Provide a minimal reproducible example
