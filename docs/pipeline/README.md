# 五阶段记忆流水线

## 概述

本文档详细描述了MemoryX的完整的五阶段记忆处理流水线，严格遵循业界最佳实践。

## 流水线阶段

```
[1. 抽取 → 2. 整合 → 3. 存储 → 4. 检索 → 5. 遗忘
```

## 各阶段文档

| 阶段 | 文档 | 描述 |
|------|------|
| 1. 抽取 | [01_extraction.md | 从原始数据中提取有价值的信息 |
| 2. 整合 | [02_consolidation.md | 去重、冲突检测、记忆关联 |
| 3. 存储 | [03_storage.md | 持久化保存与索引 |
| 4. 检索 | [04_retrieval.md | 智能检索与上下文注入 |
| 5. 遗忘 | [05_forgetting.md | 记忆生命周期管理 |

## 核心特性

- **插件化设计**: 每个阶段都可以独立配置和扩展
- **可观测性**: 每个阶段都有明确的Metrics定义
- **容错能力**: 阶段间有优雅降级机制
- **性能优化**: 每个阶段都有性能目标
- **本地优先**: 所有数据处理本地执行，数据自主可控
